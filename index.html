How to Replace HTML File in GitHub

1 Using GitHub Website (Easiest)

Go to your repository: https://github.com/forestwebpage/student-portal
Click on the index.html file
Click the pencil icon (✏️) to edit
Replace all content with the new code
Scroll down, add a commit message, and click "Commit changes"

2 Upload New File (Recommended)

Go to your repository
Click "Add file" → "Upload files"
Drag and drop your new HTML file or click to browse
Add a commit message and click "Commit changes"
If you already have an index.html file, you may want to delete it first

3 Using GitHub Desktop

Download GitHub Desktop
Clone your repository to your computer
Replace the HTML file in the local folder
Commit and push your changes from GitHub Desktop

Important Notes:

  • GitHub Pages may take 1-2 minutes to update after you make changes
  • If you have trouble, try clearing your browser cache or doing a hard refresh (Ctrl+F5)
  • Your site will be available at: https://forestwebpage.github.io/student-portal/
